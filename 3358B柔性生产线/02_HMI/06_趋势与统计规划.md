# 趋势与统计规划

## 1. 趋势页是否必要

对 3358B 这类线，趋势页不是最优先，但建议保留基础趋势和统计页，原因是：

- 教学演示需要看节拍变化
- 调试需要看站完成时间
- 分拣站和输送站适合看连续运行稳定性

## 2. 建议趋势变量

- 整线循环时间
- 01 供料周期时间
- 02 加工周期时间
- 03 搬运周期时间
- 04 装配周期时间
- 05 分拣周期时间
- 报警次数累计
- OK 数量
- NG 数量

## 3. 建议统计画面

统计页建议显示：

- 当班总产量
- OK 数量
- NG 数量
- 良率
- 当前节拍
- 平均节拍
- 各站完成次数
- 各站故障次数

## 4. 分拣站专项统计

05 站建议单独统计：

- 类别 1 数量
- 类别 2 数量
- 类别 3 数量
- 推杆动作次数
- 分拣失败次数

## 5. 教学演示建议

教学演示页可以增加：

- 当前工艺路径高亮
- 当前工件所在站
- 当前工件类别
- 当前动作名称
- 当前站步号解释
