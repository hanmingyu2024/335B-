# 公共设备块接口参数清单

## 1. FB_Cylinder

### 推荐接口

| 区域 | 名称 | 数据类型 | 说明 |
|---|---|---|---|
| Input | `CmdExtend` | `Bool` | 伸出命令 |
| Input | `FbExtend` | `Bool` | 伸出到位反馈 |
| Input | `FbRetract` | `Bool` | 缩回到位反馈 |
| Input | `tTimeout` | `Time` | 超时参数 |
| Output | `OutValve` | `Bool` | 电磁阀输出 |
| Output | `OkExtend` | `Bool` | 伸出正常 |
| Output | `OkRetract` | `Bool` | 缩回正常 |
| Output | `Fault` | `Bool` | 故障 |
| Static | `tmAct` | `TON` | 动作超时定时器 |

### 实例 DB

- `DB_Cyl_Stopper`
- `DB_Cyl_Pusher_1`
- `DB_Cyl_Clamp_1`
- `DB_Cyl_Assemble_1`
- `DB_Cyl_Sort_1`

这些都作为 `FB_Cylinder` 实例 DB 使用，不手填变量。

## 2. FB_Motor

| 区域 | 名称 | 数据类型 | 说明 |
|---|---|---|---|
| Input | `CmdRun` | `Bool` | 启动命令 |
| Input | `FbRun` | `Bool` | 运行反馈 |
| Input | `tStartTimeout` | `Time` | 启动超时 |
| Output | `OutRun` | `Bool` | 电机输出 |
| Output | `Fault` | `Bool` | 故障 |
| Static | `tmStart` | `TON` | 启动定时器 |

## 3. FB_Conveyor

| 区域 | 名称 | 数据类型 | 说明 |
|---|---|---|---|
| Input | `CmdRun` | `Bool` | 运行命令 |
| Input | `InPos` | `Bool` | 入口到位 |
| Input | `OutPos` | `Bool` | 出口到位 |
| Input | `tRunTimeout` | `Time` | 输送超时 |
| Output | `RunOut` | `Bool` | 输送输出 |
| Output | `HasPart` | `Bool` | 有料状态 |
| Output | `TransferDone` | `Bool` | 输送完成 |
| Output | `Fault` | `Bool` | 故障 |
| Static | `tmRun` | `TON` | 输送超时定时器 |

## 4. FB_Sensor

| 区域 | 名称 | 数据类型 | 说明 |
|---|---|---|---|
| Input | `RawIn` | `Bool` | 原始输入 |
| Input | `tOnDelay` | `Time` | 接通延时 |
| Input | `tOffDelay` | `Time` | 断开延时 |
| Output | `Valid` | `Bool` | 有效信号 |
| Output | `Rising` | `Bool` | 上升沿 |
| Output | `Falling` | `Bool` | 下降沿 |
| Static | `lastIn` | `Bool` | 上一扫描状态 |

## 5. 使用原则

- 设备类 `FB` 的状态记忆放在 `Static`
- 每个设备一个实例 DB
- 设备实例 DB 不手工填变量
- 先设计设备 `FB` 接口，再创建实例 DB
