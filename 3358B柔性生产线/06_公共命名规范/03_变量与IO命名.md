# 变量与IO命名

## 1. 变量总规则

变量名按“对象 + 状态/动作”组织，不写地址式命名。

推荐结构：

```text
作用域.类别.对象.状态
```

或在 `DB_Station` 内使用：

```text
In...
Out...
Man...
```

## 2. DB_Station字段命名

### 输入量

统一前缀：

- `In`

例如：

- `InTopFwd`
- `InTopBwd`
- `InPushFwd`
- `InPartAtProc`
- `InServoAlarm`

### 输出量

统一前缀：

- `Out`

例如：

- `OutTopValve`
- `OutPushValve`
- `OutClampValve`
- `OutMotorRun`

### 手动命令

统一前缀：

- `Man`

例如：

- `ManTopValve`
- `ManPushValve`
- `ManGripClamp`
- `ManPusher1`

### 状态量

统一前缀：

- `Sts` 用于 HMI 分组
- 在 DB 内部可保留具体语义名

例如：

- `StationReady`
- `StationBusy`
- `StationDone`
- `FaultActive`
- `Step`

## 3. I/O 映射命名

### 原始地址

原始地址只放在 `FC_IMap` 和 `FC_QMap` 中。

例如：

```text
DB_Station.InTopFwd := %I0.0;
%Q0.0 := DB_Station.OutTopValve;
```

### 不推荐写法

```text
Top1 := I0.0
Q0_0_Top
Input_1
```

## 4. 参数命名

时间参数统一：

- `t` 开头

例如：

- `tTopExtend`
- `tPushExtend`
- `tAlarmDelay`
- `tSortWindow`

数值或速度参数：

- 名词语义化

例如：

- `AnalogSpeed`
- `StationNo`

## 5. HMI变量别名命名

推荐格式：

```text
S01.Sts.Step
S01.IO.InTopFwd
S01.IO.OutTopValve
S01.Cmd.ManTopValve
```

整线变量统一：

- `Line.CmdStart`
- `Line.CmdStop`
- `Line.StsFault`
- `Line.TotalCount`
