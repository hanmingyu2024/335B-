# 3358B整线总工艺流程图

以下流程根据 5 个站的 IO 功能综合推断，重点体现工艺顺序和站间交接关系。

```mermaid
flowchart TD
    A[系统上电] --> B[各站初始化与原点复位]
    B --> C{自动模式且启动按钮有效?}
    C -- 否 --> B
    C -- 是 --> D[01供料站检测料仓与原料]
    D --> E[01供料站顶料并推出工件到出料位]
    E --> F[03输送站移动到供料位取件]
    F --> G[03输送站将工件送至02加工站]
    G --> H[02加工站检测有料并夹紧]
    H --> I[02加工站压头下压加工]
    I --> J[02加工站压头回上并松开工件]
    J --> K[03输送站从02加工站取件]
    K --> L[03输送站送件到04装配站]
    L --> M[04装配站检测工件与零件状态]
    M --> N[04装配站完成零件供给与装配动作]
    N --> O[03输送站从04装配站取件]
    O --> P[03输送站送件到05分拣站]
    P --> Q[05分拣站检测工件类型]
    Q --> R[05分拣站按类别推出分拣]
    R --> S{继续循环?}
    S -- 是 --> D
    S -- 否 --> T[停机待机]
```

## 站间主交接关系

- `01供料站 -> 03输送站`：供料完成、允许取料
- `03输送站 -> 02加工站`：到位放料、允许加工
- `02加工站 -> 03输送站`：加工完成、允许取件
- `03输送站 -> 04装配站`：到位放料、允许装配
- `04装配站 -> 03输送站`：装配完成、允许取件
- `03输送站 -> 05分拣站`：到位放料、允许分拣

## 说明

- `03输送站` 是整线节拍的核心搬运站。
- 总图没有展开手动、报警、复位分支，只保留主工艺路径。
