# 04站装配站 在线和诊断说明

建议重点在线查看：

- `DB_Station.Step`
- `DB_Station.InAsmPartAtPos`
- `DB_Station.InPartPresent`
- `DB_Station.InTopFwd / InTopBwd`
- `DB_Station.InSwingLeft / InSwingRight`
- `DB_Station.InHandDown / InHandUp`
- `DB_Station.InArmIn / InArmOut`
- `DB_Station.OutRotateValve`
- `DB_Station.OutGripValve`

常见诊断重点：

- 零件不足
- 旋转不到位
- 手臂和手抓动作不到位
- 装配动作未完成
