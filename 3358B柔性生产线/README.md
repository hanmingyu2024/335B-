## 使用建议

1. 先看整线说明和工艺流程图，确认五站职责。
2. 再看每站程序块结构、块接口说明和 DB 字段清单。
3. 最后根据 HMI 规划、通信点表和报警清单完善博途项目。

## 当前定位

这套资料目前定位为：

- 教学/实训级柔性产线项目模板
- 可继续深化到更完整的工程化版本

它已经适合做：

- 项目框架搭建
- 单站顺控编写
- HMI 画面规划
- 站间通信梳理

但如果要直接作为工业正式投产项目，还需要进一步补安全、异常恢复、超时处理和联锁细节。
