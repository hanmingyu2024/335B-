# 03站输送站 在线和诊断说明

建议重点在线查看：

- `DB_Station.Step`
- `DB_Station.InHome`
- `DB_Station.InLeftLimit / InRightLimit`
- `DB_Station.InLiftUp / InLiftDown`
- `DB_Station.InArmIn / InArmOut`
- `DB_Station.InGripOk`
- `DB_Comm.TransferJob`
- `DB_Comm.ReadyForJob`
- `DB_Comm.JobDone`

常见诊断重点：

- 原点丢失
- 伺服报警
- 搬运任务卡在中间步
- 取件或放件未完成
