## 2. 各站监控页最少对象

### 01供料站

- 当前步号
- 原料状态
- 挡料气缸状态
- 推料气缸状态
- 出料检测
- 与 03 站交接状态

### 02加工站

- 当前步号
- 工件到位
- 夹紧状态
- 加工头上下状态
- 加工台进退状态
- 与 03 站交接状态

### 03输送站

- 当前步号
- 原点/左右限位
- 升降、旋转、伸缩、夹紧状态
- 当前取料位/放料位
- 当前交接任务

### 04装配站

- 当前步号
- 来料状态
- 顶料、挡料、旋转、手爪、升降、手臂状态
- 装配完成信号
- 与 03 站交接状态

### 05分拣站

- 当前步号
- 分拣类别
- 推杆 1/2/3 状态
- 编码器/检测状态
- 计数和分拣结果
