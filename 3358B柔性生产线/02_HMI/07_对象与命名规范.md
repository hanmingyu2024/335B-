# 对象与命名规范

## 1. 画面命名

建议统一英文文件名 + 中文标题：

| 画面标题 | 建议文件名 |
|---|---|
| 整线首页 | `Scr_LineHome` |
| 整线总览 | `Scr_LineOverview` |
| 01供料站监控 | `Scr_S01_Monitor` |
| 01供料站手动 | `Scr_S01_Manual` |
| 报警页 | `Scr_Alarm` |
| 参数页 | `Scr_Param` |

## 2. 对象命名

建议对象前缀固定：

- 按钮：`Btn_`
- 指示灯：`Lamp_`
- 数值显示：`Num_`
- 文本：`Txt_`
- 趋势：`Trend_`
- 页面跳转：`Nav_`
- 输入框：`In_`

例如：

- `Btn_LineStart`
- `Btn_S01_ManPush`
- `Lamp_S03_Fault`
- `Num_S04_Step`

## 3. 变量命名

画面变量建议使用：

- `Line.`
- `S01.`
- `S02.`
- `S03.`
- `S04.`
- `S05.`

不要在 HMI 变量里直接写 PLC 地址式名字作为最终显示名。

## 4. 颜色规范

- 绿色：运行/正常
- 黄色：待机/注意
- 红色：故障/急停
- 灰色：未使能
- 蓝色：导航/普通操作

## 5. 交互规范

- 危险按钮使用红色或橙色
- 手动动作按钮尽量带保持提示
- 自动画面和手动画面用颜色明显区分
- 当前页导航高亮
