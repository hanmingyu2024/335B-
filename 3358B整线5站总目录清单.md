# 3358B整线 5 个站总目录清单

## 1. 说明

- 本清单用于 `3358B` 生产线的 `5` 个工位 PLC 项目统一建档。
- 当前结构采用：`整线统一模板 + 各站设备实例差异化`。
- 图中出现的 `OB123` 保留即可，但**不要当作上电初始化块使用**。
- 真正的上电初始化应使用系统启动组织块 `OB100`；如果当前站里没有 `OB100`，就另外新建一个真正的 `OB100`。

## 6. 主站与相邻站通信建议

### 6.1 主控下发命令

```text
Cmd_Auto
Cmd_Start
Cmd_Stop
Cmd_Reset
Cmd_ManualEnable
```

### 6.2 各站上传状态

```text
Sts_Ready
Sts_Busy
Sts_Done
Sts_Fault
Sts_AlarmCode
Sts_Heartbeat
```

### 6.3 相邻站交接信号

```text
Req_In
Allow_In
Req_Out
Allow_Out
Part_Present
Transfer_Done
```

## 7. OB1 推荐调用顺序

```text
FC_I映射
DB_Comm
DB_Station
DB_Alarm
FC_Q映射
```

## 8. 备注

- `OB123` 可以保留做系统异常处理，但不要代替 `OB100`。
- 如果某站设备较少，可以删除未使用的设备实例 DB，但保留统一模板框架。
- 五个站尽量保持同一命名风格，便于联调和后期维护。
